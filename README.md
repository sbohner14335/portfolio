## Getting Started
Requires having [Node.js](https://nodejs.org/en) & [pnpm](https://pnpm.io/installation) installed.

### Installation
Ensure you are using Node.js v22 or higher
```bash
nvm use
```

Install the dependencies:

```bash
pnpm install
```

### Development

Start the development server with HMR:

```bash
pnpm run dev
```
